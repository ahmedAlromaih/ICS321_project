# KFUPM Online Store

A Vite React website with a MySQL backend for the ICS321 KFUPM Online Store project.

## Overview

The system lets KFUPM users browse products, manage a cart, submit a payment form, and create orders in MySQL. Admin users can view store statistics, accounts, customer orders, and cancel orders.

It also supports seller product management, seller delivery confirmation, customer receipt confirmation, seller credit tracking, withdrawal requests, and product-specific review display.

## Features

- Sign in and sign up pages.
- Customer store with product search, cart, checkout, payment page, and order creation.
- Customer settings page with account details, account updating, order history, and a My Products seller page.
- Customer product selling with image URL, condition, price, quantity, category, and product editing.
- Seller order tracking with buyer contact links and delivery confirmation.
- Customer receipt confirmation after seller delivery confirmation.
- Seller credit / wallet card with available balance, pending balance, total withdrawn, and withdrawal request form.
- Seller withdrawal request flow with estimated arrival in 4 working days.
- Product review modal when clicking a product card.
- Out-of-stock products hidden from the customer shop page but still visible in the admin page.
- Admin dashboard for summary statistics, all orders, account creation dates, and customer order history.
- Admin order cancellation with inventory restore.
- MySQL schema auto-checks for required product selling columns, order delivery confirmation columns, seller wallet table, and withdrawal request table.
